# Support

## General Questions And Discussion

For setup help, usage questions, and general discussion, join the Valv Discord: <https://discord.gg/29a3dVRdE>

## Bugs And Feature Requests

If you have found a reproducible bug or want to propose a concrete feature, open a [GitHub Issue](https://github.com/DRNKNDev/valv/issues/new/choose) using the bug report or feature request form.

## Security Vulnerabilities

Do not report vulnerabilities on Discord or in a public issue. Use [GitHub private vulnerability reporting](https://github.com/DRNKNDev/valv/security) as described in [`SECURITY.md`](./SECURITY.md).
